
# Hospitality Analytics – End-to-End Data Analytics Project 

# Problem Statement
The hospitality industry generates large volumes of booking, revenue, and operational data, but decision-makers often struggle to convert this raw data into actionable insights.

The objective of this project is to analyze hotel booking and performance data to:
- Understand revenue and occupancy trends
- Identify high and low performing periods
- Track key hospitality KPIs
- Enable management to make data-driven business decisions

This project simulates a **real-world hospitality analytics use case** using industry-relevant tools and workflows.

---

# Project Objectives
- Perform data analysis using **SQL**
- Clean and preprocess data using **Excel**
- Build interactive dashboards using **Power BI & Tableau**
- Generate business insights related to bookings, revenue, and performance
- Demonstrate an **end-to-end analytics pipeline**

---
